# Network Science 
 
Network Science を読んでいて、使えそうなものを実装。
 
![](picture/network_science.jpg)
 
## Requirement

* collections
* matplotlib
* networkx
* numpy
* scipy
* seaborn


## Installation

with Python 3.6+

```sh
$ pip install -i https://test.pypi.org/simple/ network-science
```

ちゃんと整えたら本番PyPIにアップします。
